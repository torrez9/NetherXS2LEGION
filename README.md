# NetherSX2-patch
Esta es una continuación de NetherSX2 para construir sobre el increíble trabajo ya realizado por Anon y EZOnTheEyes


Los objetivos son los siguientes:
* Eliminar la carga innecesaria de servicios de anuncios que quedó en el apk
* Arreglar las Notificaciones de RetroAchievements
* Exponer más configuraciones globales de la aplicación al usuario
* Actualizar la GameDB, Soporte de Controles, y los Parches de Pantalla Panorámica y Sin Entrelazado
* Añadir correcciones específicas de AetherSX2/NetherSX2 adicionales a la GameDB
* Re-firmar el APK para eliminar la advertencia de Play Protect


## Instalando NetherSX2
Una vez que hayas obtenido una copia del [NetherSX2 APK](https://github.com/Trixarian/NetherSX2-patch/releases/download/2.1/NetherSX2-v2.1-4248.apk), es momento de instalarlo en tu dispositivo


Si estabas usando una copia de NetherSX2 1.9a o superior: 
1. (Opcional) Haz un respaldo de tus archivos en caso de que algo salga mal. El método más fácil es usar la función integrada de Transferir Datos de AetherSX2 usando su característica de Exportar para mover tus archivos a una carpeta externa. Esto respaldará tus archivos de bios, tarjetas de memoria, estados guardados, configuraciones de juegos, carátulas y paquetes de texturas
2. Instala el apk normalmente para actualizar a la última versión y mantener tu configuración actual


Si estabas usando AetherSX2 o una copia de NetherSX2 anterior a 1.9a:
1. Respalda tus archivos. El método más fácil es usar la función integrada de Transferir Datos de AetherSX2 usando su característica de Exportar para mover tus archivos a una carpeta externa. Esto respaldará tus archivos de bios, tarjetas de memoria, estados guardados, configuraciones de juegos, carátulas y paquetes de texturas
2. Después de respaldar tus archivos, necesitas eliminar cualquier copia anterior de AetherSX2 o NetherSX2. Haz esto desinstalando la aplicación normalmente. Asegúrate de NO mantener ningún archivo si se te solicita
3. Con el respaldo y las desinstalaciones completadas, todo lo que queda es navegar a donde pusiste tu apk de NetherSX2 con tu Administrador de Archivos (llamado Archivos o Administrador de Archivos dependiendo de tu dispositivo) y tocarlo para instalarlo en tu dispositivo
4. Una vez instalado, ejecuta la aplicación y configúrala normalmente. Una vez en la pantalla de Lista de Juegos, puedes acceder a la función de Transferir Datos/Respaldar Datos e Importar los archivos que exportaste anteriormente navegando a la carpeta donde pusiste todos tus archivos respaldados. Esto debería importar tus archivos de bios, tarjetas de memoria, estados guardados, configuraciones de juegos, carátulas y paquetes de texturas
5. Ahora dedica algo de tiempo a rehacer la Configuración Global de la Aplicación y deberías estar listo para comenzar


## Preguntas Frecuentes
### ¿Por qué usar NetherSX2 en lugar de AetherSX2?
Usa NetherSX2 si quieres:
* RetroAchievements
* Archivos de configuración actualizados
* Correcciones de errores específicas de NetherSX2 para juegos
* Mejor soporte de controles para configuración automática
* Tener más control sobre tu emulador mediante configuraciones


### ¿Qué variante debo usar entre Classic (3668) y Patched (4248)?
Esto depende en gran medida de los juegos que vayas a jugar. Generalmente Classic será más eficiente - y algunos juegos solo funcionarán bien en él como los de Nascar - pero le faltarán algunas correcciones gráficas y de errores solo presentes en 4248. Se recomienda usar 3668 con Mali o dispositivos menos potentes para obtener el mejor rendimiento. Para dispositivos más potentes o con Adreno, se recomienda usar la variante 4248


### ¿Cuáles son las mejores configuraciones para el rendimiento?
Las configuraciones predeterminadas de Óptimo/Seguro son las más compatibles y eficientes para la mayoría de juegos. Puedes optimizarlas aún más usando el renderizador de GPU Vulkan y configurando el Modo de Descarga de Hardware en Deshabilitar Lecturas. Con 3668, también asegúrate de habilitar Presentación con Hilos cuando uses el renderizador de GPU Vulkan
